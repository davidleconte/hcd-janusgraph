# Banking Use Cases Implementation

This directory contains the implementation of 4 complex banking use cases leveraging the HCD + JanusGraph + OpenSearch stack.

## Use Cases

1. **Anti-Money Laundering (AML)** - Network detection and compliance
2. **Fraud Rings Detection** - Coordinated fraud and insider threats
3. **Customer 360** - Personalized insights and recommendations
4. **Trade Surveillance** - Market manipulation and insider trading

## Directory Structure

- `schema/` - Graph and vector schemas
- `data/` - Sample data generators
- `queries/` - Gremlin queries for each use case
- `notebooks/` - Jupyter notebooks with examples
- `docs/` - Detailed documentation
