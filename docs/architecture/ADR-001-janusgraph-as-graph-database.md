# ADR-001: JanusGraph as Graph Database

**Status:** Accepted  
**Date:** 2025-01-15

## Context
Need a scalable graph database for banking fraud detection and AML compliance.

## Decision
Use JanusGraph with HCD (Hyper-Converged Database) backend.

## Consequences
- Scalable graph storage with Cassandra-compatible backend
- Native Gremlin query support
- Integration with existing IBM infrastructure
